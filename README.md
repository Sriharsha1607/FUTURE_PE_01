# FUTURE_PE_01 – AI Website Copy Generator

## Project Overview

This project demonstrates how Prompt Engineering can be used to generate high-quality website copy for local businesses.

Many small businesses struggle with unclear messaging, weak value propositions, and ineffective calls-to-action. Hiring professional copywriters can be expensive.

This project solves that problem by designing a **structured prompt system** that uses AI to generate clear, persuasive, and conversion-focused website content.

---

## Business Chosen

Brilliant’s Coaching Center
Location: Vijayawada, India

This coaching institute provides academic support for students preparing for board exams and competitive tests.

---

## Objective

Design a reusable prompt framework that can generate:

- Homepage copy with a clear value proposition
- Services page descriptions
- Persuasive call-to-action (CTA) sections
- Tone-adaptable content for different business types

The generated content should be **ready for real website publishing**.

---

## Prompt Engineering System

The prompt system uses structured inputs:

* Business Name
* Business Type
* Location
* Target Audience
* Services Offered
* Tone of Voice

Using these inputs, the prompt generates structured website content.
